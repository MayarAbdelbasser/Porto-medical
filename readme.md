# Medical Website

A modern and responsive medical website built with HTML, CSS, SASS, and JavaScript. This project is designed to showcase medical services, doctors, departments, resources, and insurance providers.

## Overview

This website serves as a template for a medical or healthcare website. It is fully responsive and provides an interactive UI for users to navigate departments, view doctors, and access resources.

## Features

- **Responsive Layout:** Works on all devices including desktop, tablet, and mobile.
- **Header & Navigation:** Sticky header with navigation links and contact info.
- **Hero Section:** Attractive hero banner with medical highlights.
- **About Section:** Information about the medical facility.
- **Departments Section:** Cards showcasing different medical departments.
- **Doctors Section:** Carousel to navigate through doctorsâ€™ profiles.
- **Resources Section:** Cards displaying features or resources.
- **Insurance Section:** Logos of accepted insurance providers.
- **Quote Section:** Testimonial section with a quote.
- **Footer:** Contact information, opening hours, emergency cases, and social media links.
- **Interactive Carousel:** Navigate through doctors using arrow controls.

## Technologies Used

- HTML5
- CSS / SASS
